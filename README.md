# SAFEcam

25.01.21
처음에 이 프로젝트를 구상했을 때는 생성형 ai만 가지고 개발을 진행해보려 했었다. 하지만 계속해서 막히는 문제점들을 해결하기엔 벅찼고, 결국 Apple이 공식으로 준 샘플코드를 차근차근 분석 및 swift 문법 공부를 통해 당초 계획했던 앱을 30일 내로 만드는 것이 목표이다. 현재 Camera 파일과 SAFEcamApp에 대한 이해는 마쳤다. Camera 파일 자체는 일종의 객체이자 클래스라고 보면 될 것 같다. 이걸 참조해서 CameraViewmodel을 만들 것 같다. SAFEcamApp은 앱의 가장 기본이다. 이 부분은 플러터로 앱을 개발했을때와 유사하다.
